My first blog

第一篇个人blog，记录一下在win10使用hexo搭建个人blog的过程。

参考

感谢大大们的分享：
b站up主的hexo教程：https://www.bilibili.com/video/BV1Yb411a7ty?from=search&seid=14978670385120800724

一、前期环境搭建

1.安装并配置好node.js;

2.安装并配置好git：https://git-scm.com/download/win;

3.Github官网注册获取username以及email;

二、测试环境

1.win10命令行（win+R+cmd）：

输入命令：
node -v
npm -v
会出现对应的node和npm的版本信息

三、开始安装

1.安装cnpm（淘宝镜像源）;

npm install -g cnpm --registry=http://registry.npm.taobao.org

2.安装hexo框架;

cnpm install -g hexo-cli

四、生成第一篇blog

1.创建blog文件夹，cd进入

mkdir blog
cd blog

2.初始化博客，生成默认的初始blog

hexo init
如果这一步提示“’git’不是内部或外部命令，也不是可运行的程序或批处理文件
则需要给git配置环境变量(如果还没下载的话先下载)，再用win命令行重新试一下；
如果还不行，则直接打开git bash命令行输入：hexo init

3.启动本地blog服务

hexo s
s是start简写
关闭服务：Ctrl+c 注意：关闭服务后浏览器不可以再通过第4步进行访问。

4.浏览blog

浏览器输入localhost:4000本地访问生成的blog（一般端口是4000，
不知道有没有特殊的）

5.写第一篇blog

hexo n "My first blog"
n是new的简写 新建
ls
浏览文件目录下的文件
vim My-first-blog.md
打开进行编辑(即刚刚新建的’My first blog’的文件)

6.重新开启hexo并刷新浏览

cd ../..
返回blog目录下
hexo clean
清理一次
hexo g
g是generate简写，重新生成一下
hexo s
重新启动hexo服务，刷新浏览器可以看到新编辑的blog

五、部署到远程Github仓库

1.登录Github官网，新建一个仓库

登录后，点击右上角**+号，
**New repository–>Repository name下方空格填写’YourGithubName.github.io’
注意：YourGirhubName指的是你自己Girhub账号的用户名
随后点击下方的Create repository，这样仓库就已经创建成功
创建成功后，在Quick setup一栏中有你的仓库链接
如“https://github.com/YourGithubName/YourGithubName.github.io.git”

2.blog文件夹部署

cnpm install --save hexo-deployer-git
在blog目录下安装git的插件
vim _config.yml
安装插件后会有一个yml文件，用vim进行编辑（直接在文件夹中用记事本方式打开也可）
拉到文件最下方
修改Deployment


注意空格
修改完成后保存退出

3.部署到Gitnub

hexo d
d是deploy的简写，就这一条命令即可
过程中需要输入YourGithubName以及password密码，可能是在命令行输入，也可能是直接调出Github的登录界面

4.浏览器刷新Github仓库

发现仓库已经有我们本地的数据

六、通过域名访问个人博客

网址是：“https://YourGithubName.github.io”
也可以在Github上找到个人博客网址
Your repositories–》选择仓库YourGithubName.github.io–》Settings–》在Github Pages那里可以看到：“Your site is published at https://YourGithubName.github.io/”。

七、遇到的坑

1.本地博客可以通过浏览器localhost:4000成功浏览，也可部署到Github上，但是打开Github Pages网址时总是无法连接到服务器。
DNS服务器解析错误，修改DNS服务器地址：
打开控制面板–》网络和Internet–》网络和共享中心–》点击正在用的如网络“连接：WLAN”
–》在常规这看看连接Internet的是IPv4还是IPv6（我的是IPv4）–》属性–》选定Internet 协议版本4
–》属性–》修改DNS服务器地址为：114.114.114.114–》保存
浏览器刷新就可了
